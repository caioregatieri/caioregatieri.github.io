Sobre mim

Eu amo codificar, começei a programar no ensino médio e desde então eu não parei. Formei-me em técnico em informática, depois me graduei em análise e desenvolvimento de sistemas e então me especializei em desenvolvimento web (Java e .Net).
Fiz e ainda venho fazendo outros diversos cursos rápidos na area de desenvolvimento de aplicativos web e móveis.

Eu começei em 2005 criando aplicativos para computador, desenvolvendo-os com VB6, depois migrei para o C# onde desenvolvi aplicativos mais complexos e então começei a desenvolver para web com ASP .Net, em seguinda conheci o PHP e seus diversos frameworks, dentre eles Laravel e CakePHP, com os quais criei alguns projetos muito legais.
Pouco depois com a grande demanda por aplicativos para smartphones eu começei a desenvolver aplicativos híbridos usando técnologias baseadas em Javascript (Cordova e Ionic).
Atualmente venho trabalhando e estudando o React Native para desenvolvimento mobile.

Se você precisa desenvolver um aplicativo ou um site conte comigo.

Mavi Click

Catalogo de produtos

Sistema de catalogo de produtos

Linguagens: JS (Node e AngularJS)

Banco de dados: Mysql

Esse projeto foi realizado por inteiro, tendo sido desevolvido o backend em nodejs rodando 3 aplicações para evitar gargalos e melhorar a performance, e o frontend com Angular. O projeto conta com cadastros de usuarios, clientes, produtos, marcas, fornecedores, setores, arquivos, importação de xls para cadastro automatico de produtos, importação de zip contendo imagens e arquivos vinculados a produtos, da mesma forma possui a opção de exportar esses mesmos tipos de arquivos com base nos produtos. Possui controle de acesso e log de ações, produtos relacionados a clientes espeficios, gerenciamento de permissões. O banco de dados possui algumas tabelas com mais de um milhão de registros e a aplicação foi desenvolvida para manipular esses dados de forma que consegue retornar uma consulta com mais de 200 mil registros em segundos. O sistema conta tambem com agendamento de backup automatico dos dados e otimização de uso de armazenamento de dados e protocolo de segurança SSL.
